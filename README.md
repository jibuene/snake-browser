# JIB-Snake

Basic snake game written in VUE.js

### Prerequisites

Node


### Installing
```
npm install
```
```
npm run serve
```
## Built With

* [VUE.js](https://vuejs.org) - JavaScript Framework
* [Vue-Konva](https://github.com/konvajs/vue-konva) - Konva for VUE
